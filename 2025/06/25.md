## わかったこと
- git のコマンドでcheckoutは様々な用途があるコマンド
    - 変更の取り消し
    - ブランチの切り替え
- git pullとpushについて理解が深まった
    - pull 読み取りの権限のみで実行可能
    - push書き込みの権限が必要だから危険
    このような理由から、pushリクエストではなくpullリクエストとなっている
- git frowとgit hub frowについて少し理解できた
- forkとcloneについて違いが分かった
